# hollow_knight 前端项目

## 项目简介
一个以《空洞骑士》为主题的前端项目，具体展示空洞骑士的相关内容（如地图、图鉴、美术）。

## 技术栈
前端框架：Vue3

构建工具：Vite

## 安装和运行

1.克隆仓库

```bash
git clone https://github.com/oodrain/hollow_knight.git
cd hollow_knight
```
2.安装依赖
```bash
npm install
```

3.本地运行
```bash
npm run dev
```
